Case Study: A Responsive and Accessible Header

The objective was to create a header component for BanjoDave.com, a React application sharing my hobby projects. The header needed to provide clear and straightforward navigation across different sections of the website, adapt to different screen sizes, and meet accessibility standards for inclusive user experience.

View Header.js on GitHub

  • Adaptive Dropdown Menu: In order to cater to devices with smaller screen real estate, I designed a dropdown menu that accommodates secondary navigation links. This menu unveils its content when the user hovers over it. I've included specific adjustments for users who interact with the dropdown by clicking. To prevent the dropdown from abruptly closing just after it opens, I implemented a feature that disables response to user clicks for a specific duration after the dropdown opens or closes via a hover action.
  • Navigation Links: The website's navigation links are fetched from an external file menuItems.js. This separation of concerns ensures that changes to navigation items can be managed without interfering with the Header component's core functionality.
  • Accessible Navigation: To ensure the website is accessible, I included a 'skip-link' that allows users to skip to the main content of the website. This feature is particularly useful for users who navigate with a keyboard rather than a mouse.

Case Study: A Dynamic and Interactive Tile Component

The primary goal was to develop a compelling, interactive component for showcasing hobby projects on BanjoDave.com. This component needed to capture visitors' attention and encourage them to explore further

View SkewDiv.js on GitHub

  • Dynamic Polygon Clipping: The SkewDiv component incorporates a breathing-like animation by using dynamic polygon clipping with CSS clip-path property. The clip-path is created through a function, generateClipPath, that utilizes an array of random offsets generated with JavaScript's Math.random function. The polygon shape defined by the clip-path provides a distinctive look for each tile and changes subtly every 2 seconds through a setInterval function.
  • Hover Interactivity: User interactivity is enhanced through hover effects. When a user's mouse enters the tile (triggered by onMouseEnter), two animations are initiated. Firstly, the clip-path of the tile undergoes changes, giving a pulsating effect. This effect is achieved by the startInterval function that sets the animation interval. Secondly, a wave-like animation is applied to the text within the tile, where each character in the title is assigned an animation delay based on its index, created by mapping over the title string.
  • Versatility and Reusability: The SkewDiv component accepts various properties, such as imgSrc, dest, title, and description, enhancing its reusability across different projects. Thus, it presents an engaging and unique appearance while eliminating the need for custom coding for each new project.

Case Study: Visually engaging parallax scrolling effect

In designing the Outback Adventures homepage, the key goal was to create an engaging user experience reminiscent of a journey through the Australian outback. Utilizing a parallax scrolling effect, the page simulates a 3D landscape, achieved by carefully manipulating SVG images of hills and adjusting their scales based on their assigned 'depth'. The incorporation of dynamic elements like an animated sun and shifting clouds further elevates the immersive feel of the website. Critical to this design was the use of CSS transforms, particularly the `translateZ` function and `transform-style: preserve-3d` property, effectively creating a sense of depth and making the website's landscape come alive.
