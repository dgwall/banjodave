Map Modifiers

Some .exe programs written in Processing, to apply modifications to maps for n v1.4.

Red Gold

This isn't a program but check out what happens if you replace all the gold in N with mines. If you want to do this to maps yourself, just run this find and replace on the map data: Find [!0^], replace with [!12^]. All of the v1.4 Metanet maps with RED GOLD applied can be found on packs.droni.es.

Chaos Edition

This program will add random NReality mods to each map given to it. Paste your maps into the input.txt file, run the program, then play. Requires NReality of course.

Corruption

I wrote this program to corrupt a set of maps. It takes the tiles and items and shifts the bits of data up or down one. This feature is also a part of Chaos Edition. Creates some interesting challenges and glitches. Occasionally will create maps that crash N, so be aware of that.
